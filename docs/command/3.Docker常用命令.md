|序号|命令描述|命令示例(可双击复制)|可选参数|
|:--:|:--|:--|:--|
|01|查看正在运行的容器|`docker ps`|`-a`:重看所有包括已停止的容器|
|02|删除所有旧的，未使用的镜像、容器、网络和构建缓存(==慎用=={.warning})|`docker system prune -a`|`-f`:跳过二次确认|
|03|删除悬空(dangling)镜像|`docker image prune`|`-f`:跳过二次确认, `-a`:包含所有未使用的镜像|
