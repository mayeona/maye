|序号|命令描述|命令示例(可双击复制)|可选参数|
|:--:|:--|:--|:--|
|01|查看正在运行的容器|`docker-compose ps`|`-a`:查看所有包括已停止的容器|
|02|创建docker-compose.yaml中定义的所有容器和网络并启动|`docker-compose up`|`-d`:以后台方式启动；`--force-recreate`:重建容器；`--build`:重建镜像|
|03|停止并删除容器、网络|`docker-compose down`|`-v`:同时删除数据卷|
|04|仅从dockerfile构建镜像|`docker-compose build`|`up -d`:同时后台启动并运行|
