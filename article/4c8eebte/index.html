安装KubeSphere

约 232 字小于 1 分钟

K8sKubeSphere

2022-08-28

前言

市面上开源的 k8s 管理平台有不少，平时工作中用的是 Rancher2，对比 kubeSphere 后了解到，前者更适合运维操作，后者对开发友好，对比之后决定安装 kubeSphere。

1. 安装 kubeSphere

从官网 找到教程，使用all in one的方式安装。

  1. 确保 DNS /etc/resolv.conf 可用，最好先关闭防火墙
  2. 安装依赖 apt-get install [socat|conntrack|ebtables|ipset]
  3. 下载 KubeKey，可先从 github 上看最新的 release 版本。
  4. 在 root 账户，cd /home/maye执行：export KKZONE=cn 和 curl -sfL https://get-kk.kubesphere.io | VERSION=v2.2.2 sh -

2. 登录控制面板

虚拟机 ip 是 192.168.10.100，端口是 30880，但由于使用的是 nat 模式，开启了端口映射，所以访问地址是：127.0.0.1:30880 。初始账号密码为：admin/P@88w0rd , 已修改为 Mayee@66

网络模式的选择有待调整。
